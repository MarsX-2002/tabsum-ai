TabSum AI Demo Recording Guide

Recording Steps:

  1. Install and configure TabSum AI extension
  2. Open a complex article (e.g., Wikipedia)
  3. Click the TabSum AI icon
  4. Select different summary lengths
  5. Show the dark mode toggle
  6. Demonstrate sharing options
  7. Show the copy feature

Recommended Recording Settings:

  • Resolution: 1280x720
  • Frame Rate: 30fps
  • Duration: 15-30 seconds
  • Format: GIF or MP4

Recommended Recording Tools:

  • ScreenToGif (Windows, free)
  • LICEcap (Windows/Mac, free)
  • Windows Game Bar (Windows 10+, built-in)

After recording, place the GIF in the docs/demo/ folder as demo.gif
